# China Policy Risk Analysis Skill

这是一个面向普通人的中国政策分析 Skill，用于在阅读公开政策文本时，帮助识别政策目标、传导链条、执行风险、配套政策缺口，以及普通居民可能面对的利益和风险。

本 Skill 的定位是审慎分析工具，而不是立场表达工具。它强调从公开信息出发，尽量把政策影响拆解为可观察、可验证、可复盘的路径。

## 适用场景

- 分析财政、货币、金融、房地产、资本市场、养老、社保、产业、地方债、平台经济等政策。
- 判断政策可能利好哪些机构、行业或群体，可能由哪些群体承担成本。
- 评估政策落地难度、执行偏差和“看起来明确、实际落地不及预期”的风险。
- 搜索并梳理相关配套政策，判断主政策是否具备完整传导条件。
- 从普通居民视角评估就业、储蓄、负债、房产、养老金、基金、保险和其他金融产品风险。

## 分析原则

1. 不把政策文本等同于政策结果。先看政策说什么，再看钱从哪里来、风险往哪里去、谁有退出权。
2. 不只看受益方，也看受损方和最终风险承接方。
3. 不只看主政策，也看配套政策、实施细则、资金来源、监管规则和地方执行能力。
4. 不把“政策支持”理解为个人收益保证。系统性稳定工具通常保护系统流动性，不保证个人资产不亏损。
5. 普通居民建议以保守为先：先控制不可逆损失，再讨论可能的政策红利。

## 标准分析框架

使用本 Skill 时，建议按以下结构输出：

1. **核心判断**：给出一句话结论，并标注政策落地风险。
2. **政策表层与真实约束**：区分官方叙事、现实压力和可能的制度目标。
3. **传导链条**：说明政策如何影响金融机构、企业、地方政府和居民部门。
4. **受益方、受损方、风险承接方**：避免只写利好，不写成本。
5. **配套政策与缺口**：列出已有配套；若缺失，分析未来可能补哪些工具及副作用。
6. **短期影响**：关注流动性、资产价格、就业预期、融资成本和市场情绪。
7. **长期后果**：关注债务、财政、产业竞争、居民财富结构和制度约束。
8. **普通居民行动框架**：先写风险规避，再写可能机会。

## 普通居民风险提示

本 Skill 对普通居民的建议默认采取谨慎立场：

- 保留足够现金流，不为政策主题加杠杆。
- 不把理财、基金、保险、养老金产品等同于存款。
- 警惕“稳健”“养老”“政策支持”“高股息”“固收+”等营销标签。
- 长债基金、信用债基金、红利资产、主题基金都可能出现本金回撤。
- 个人养老金账户需要关注锁定期、税收适配度、产品费用和未来规则变化。
- 房产需区分自住和投资，政策托底不等于价格上涨或流动性恢复。
- 对复杂、封闭、难退出、收益承诺模糊的产品保持克制。

## 使用方式

在 Codex 中可以这样调用：

```text
Use $china-policy-risk-analysis to analyze this Chinese policy's hidden motives, unfinished-policy risk, and ordinary resident risk controls.
```

也可以直接提出中文请求，例如：

```text
用 china-policy-risk-analysis 分析这项政策对普通居民、银行、企业和地方政府的影响，并评估落地风险。
```

## 审慎边界

本 Skill 只基于公开资料和一般政策分析方法提供结构化研判，不提供个性化投资建议、法律意见或规避监管建议。对于金融产品、房产、养老金账户等事项，使用者仍需结合自身现金流、负债、家庭责任、风险承受能力和当地规则独立判断。
